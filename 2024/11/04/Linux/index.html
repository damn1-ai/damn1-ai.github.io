一、命令基础格式

​ 通用格式：command [-option] [parameter]

​ Command是命令本身 -options（可选，非必填）命令的选项，可以通过选项控制命令的行为细节。 parameter （可选，非必填） 命令的参数。多数用于命令的指向目标

​ 语法中的[ ]，表示可选的意思。

二、Linux常见命令


cd命令

1.概念

​ 用来更改用户所在的工作目录

2.语法

​ （1）cd+文件路径 表示切换到目标目录

​ （2）直接执行cd目录表示回到HOME目录


ls命令

1.概念

​ ls命令可以将当前目录下的内容列出

2.语法

​ （1）ls -a 列出全部文件（包含隐藏的文件和文件夹）

​ （2）ls -l 将文件竖向排列并展示文件更多具体信息

​ （3）ls -h 输出更易阅读的形式，文件的大小。（必须搭配 -l 命令一起用）


pwd命令

1.概念

​ 查看当前工作目录

2.语法

​ 直接输入 pwd 即可


which命令

1.概念

​ 查看程序文件存放在哪里

2.语法

​ which 要查找的命令（例如 which cd）


cp命令

1.概念

​ 可以用于复制文件/文件夹

2.语法

​ cp [-r] 参数1 参数2

​ -r 可选，表示递归（将指定目录下的所有文件与子目录一并处理）

​ 参数1，Linux路径，表示被复制的文件或文件夹

​ 参数2，Linux路径，表示复制到的位置


cat命令

1.概念

​ 可查看文件内容

2.语法

​ cat Linux路径

​ 查看此路径下的文件


echo命令

1.概念

​ 使用此命令可以在命令行内输出指定内容

2.语法

​ echo 输出的内容


示例：输入：echo HELLO WORLD

       输出：HELLO WORLD

find命令

1.概念

​ 按文件名称查找文件

​ 按文件大小查找文件

2.语法

按文件名称

​ find 起始路径 -name”被查找文件名”（被查找文件名支持通配符进行模糊查询）

按文件大小

​ find 起始路径 -size +|- n[kMG]

​ · + -表示大于和小于

​ · n表示大小数字

​ · kMG表示大小的单位，k（小写）表示kb，M表示MB，G表示GB

​ 示例：

​ · 查找小于10kb的文件： find / -size -10k

​ · 查找大于100MB的文件： find / -size +100M


grep命令

1.概念

​ 从文件中通过关键字过滤带有此关键字的行

2.语法

​ grep [-n] 关键字 文件路径

​ -n 可选，表示在结果中显示匹配的行的行号

​ 关键字 必填，建议用“ ”将关键字包围

​ 文件路径 必填，表示要过滤内容的文件路径 （此结果可以作为内容输入端口 用于管道符）


mkdir命令

1.概念

​ mkdir命令可以创建新的目录（即文件夹）

2.语法

​ mkdir [-p] Linux路径

​ -p 可选写，表示创建不存在的父目录，适用于创建多层级目录。

​ 加 -p 后可以一次性创建多个层级的目录。


more命令

1.概念

​ 类似于cat命令，但如果文件内容过多，可以一页页展示（通过空格翻页，q退出查看）

2.语法

​ more Linux路径


mv命令

1.概念

​ 移动文件或文件夹

2.语法

​ mv 参数1 参数2

​ 参数1，Linux路径，表示要移动的文件或文件夹

​ 参数2，Linux路径，表示要移动去的地方，如果此地方不存在，则进行改名，确保目标存在


rm命令

1.概念

​ 删除文件、文件夹

2.语法

​ rm [-r -f] 参数1 参数2 …..参数N

​ -r 同cp命令的-r

​ -f 表示强制删除（不会弹出确认提示）

​ ·普通用户删除内容一般不会弹出提示，只有root会

​ 参数1….N 表示要删除文件或文件夹的路径，按空格隔开


​ （支持通配符）例如 rm -r test*


tail命令

1.概念

​ 可以查看文件尾部内容，跟踪文件的最新更改

2.语法

​ tail [-f -num] Linux路径

​ -f 表示持续跟踪

​ -num 表示查看尾部多少行，不填默认10行

​ Linux路径表示被跟踪文件的路径


touch命令

1.概念

​ 通过touch命令创建文件

2.语法

​ touch Linux路径

​ 相对、绝对、特殊路径符都可以使用


wc命令

1.概念

​ 统计文件的行数、单词数量

2.语法

​ wc [-c -m -l -w] 文件路径

​ -c 统计bytes数量

​ -m 统计字符数量

​ -l 统计行数

​ -w 统计单词数量文件路径

​ 被统计的文件，可作为内容输入端口（用于管道符）


特殊路径符

​ . 表示当前目录。例如：cd ./Desktop 表示切换到当前目录下的Desktop目录内，和cd Desktop效果一样

​ .. 表示上一级目录，例如：cd .. 即可切换到上一级目录，cd ../.. 切换到上二级目录

​ ~ 表示HOME目录，例如：cd ~ 即可切换到HOME目录


相对路径和绝对路径

​ 1.相对路径：以当前目录为起点，无需以 / 开头。

​ 写法： cd Desktop

​ 2.绝对路径：以根目录为起点，需以 / 开头。

​ 写法： cd /home/damn/Desktop
