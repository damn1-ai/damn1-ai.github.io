3. limit offset绕过

这是一种绕过提取数据限制的方法,和group_concat(将多行数据合到一起用逗号隔开) , limit0 , 1(提取指定行的数据)作用类似,若这两种方式都被过滤,则可以使用limit offset绕过

格式:

limit row_count offset offset
row count 确定返回的行数
offset 偏移量，就是从第几行开始返回

示例:

payload

4. select子查询

select *from((select 1)A,(select 2)B);

派生表:

MySQL在进行子查询时，就会产生一个派生表，他是select语句产生的一个虚拟表,与外部没有关系，产生的结果传给外部select查询

示例:

select (select 1)A,(select 2)B;

A和B表示派生表的名字,可以任取,但是不能不取,不然会报错

5. join绕过

Join 用来把多个表连接起来

示例:

select * from((select 1)A join (select 2)B join (select 3)C);
